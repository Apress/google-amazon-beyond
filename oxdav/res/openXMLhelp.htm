_Oxdav Project help file

This is the file openXMLhelp.htm. This project is for managing hierarchical files such as this one, or hierarchy.html. This is a link to google.

_Data Entry

Save a strictly hierarchical file, h1-h2-&c, file into the /oxdav (webdav) webapp; look for its echo in /oxdav/res.

_Data Entry Tricks

_Copying self-links

If you're browsing this document, you can right-click any "_" anchor and copy the link location, then paste it into another document you're working on.

_MSWord base url Warning

MSWord may munge the relative URLs in your file; make sure that you set the base URL in the document properties to /oxdav/res, not just to oxdav.
