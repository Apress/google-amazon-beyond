This form does an Amazon keyword search as a GET, with the (public) URL of an XSLT page as a parameter. The URL is to be: 

http://xml.amazon.com/onca/xml2
      ?t=webservices-20
      &dev-t=[developer’s token goes here]
      &KeywordSearch=[subject keyword goes here]
      &mode=[product line goes here]
      &type=[lite or heavy]
      &page=[page # goes here]
      &f=[URL pointer to XSL document]

developer's token: Please use your own!
